Одиниці вимірювання в HTML та CSS

Що таке одиниці вимірювання

Одиниці вимірювання в HTML та CSS використовуються для визначення розміру елементів на веб-сторінці. Вони дозволяють керувати шириною, висотою, відступами, шрифтами та іншими параметрами дизайну.

Існують абсолютні та відносні одиниці вимірювання. Абсолютні мають фіксоване значення, а відносні залежать від розміру екрана або інших елементів.

Одиниці вимірювання

  • px — пікселі. Абсолютна одиниця вимірювання, часто використовується для точного контролю розмірів елементів.
  • pt — points (пункти). Використовуються переважно для друку або типографіки.
  • pc — pica. Рідко використовується одиниця вимірювання, що застосовується у друкарській справі.
  • in — дюйми. Абсолютна одиниця, іноді використовується для друкованих макетів.
  • cm — сантиметри. Абсолютна одиниця довжини.
  • mm — міліметри. Також абсолютна одиниця вимірювання.
  • q — чверть міліметра. Використовується дуже рідко.
  • % — відсотки від розміру батьківського елемента.
  • em — відносна одиниця, залежить від розміру шрифту поточного або батьківського елемента.
  • rem — відносна одиниця, що залежить від розміру шрифту кореневого елемента HTML.
  • vw — 1% від ширини вікна браузера.
  • vh — 1% від висоти вікна браузера.
  • vmin — 1% від меншого значення між шириною та висотою екрану.
  • vmax — 1% від більшого значення між шириною та висотою екрану.
  • ch — одиниця, що базується на ширині символу "0" у поточному шрифті.
